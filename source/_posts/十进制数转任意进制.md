# 十进制数转任意进制

有时候碰到进制转换问题，这里提供一个公共方法，十进制转任意进制

```c++
// 将十进制数字转换为指定进制（2-36）
QString toBase(int number, int base)
{
    if (number == 0) return "0";
    if (base < 2 || base > 36) return QString::number(number);
    static const char digits[] = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    QString result; 
    while (number > 0) {
        result.prepend(digits[number % base]);
        number /= base;
    }
    return result;
}
```

具体实现思路例子：

```tex
number = 25, base = 16

第一次循环:
  25 % 16 = 9 → digits[9] = '9'
  25 / 16 = 1
  result = "9"

第二次循环:
  1 % 16 = 1 → digits[1] = '1'
  1 / 16 = 0
  result = "19" (prepend所以是"1" + "9")

结果: "19"
```
